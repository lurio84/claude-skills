name: grill-me
description: Interview the user relentlessly about a plan or design until reaching shared understanding, resolving each branch of the decision tree. Use when user wants to stress-test a plan, get grilled on their design, or mentions "grill me" / "entrévistame".
---

Entrevista a Lucas sin piedad sobre cada aspecto de este plan hasta alcanzar entendimiento compartido.

Recorre cada rama del árbol de decisiones, resolviendo dependencias entre decisiones una a una. Para cada pregunta, da tu recomendación.

**Reglas:**
- Una pregunta a la vez. No agrupes.
- Si una pregunta se puede responder leyendo código, vault, memoria, o cualquier otra fuente disponible — hazlo tú en lugar de preguntar.
- Marca el tipo de afirmación cuando aplique (`[DATO]` / `[OPINIÓN]` / `[INTUICIÓN]`) — coherente con [[feedback_etiquetar_certeza]].
- No hagas la pregunta si la respuesta no cambia el plan materialmente. Para.
- Cuando termines, resume las decisiones tomadas en bloque para que Lucas valide de un vistazo antes de implementar.
